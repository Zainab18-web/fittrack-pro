# 💪 FitTrack Pro

A complete Health & Fitness Tracker web app built with React.

## Features
- 💧 Water Intake Tracker
- 🏋️ Workout Logger
- 🍎 Calorie Tracker
- ⚖️ Weight Progress (with charts)
- 😴 Sleep Tracker
- 🍽️ 7-Day Meal Planner
- 📸 Progress Photos
- 🔔 Reminders & Notifications
- 📊 Weekly/Monthly Reports

## Tech Stack
- React 18 + Vite
- Recharts (data visualization)
- localStorage (data persistence)

## Run Locally
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
```
Then upload the `dist` folder to any static host (Vercel, Netlify, etc.)
